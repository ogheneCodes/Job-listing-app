# 💼 Job Listing Application

A simple Job Listing Application built with React and TypeScript.  
This project displays job listings using reusable React components and props.

---

# 🚀 Features

- Display job listings
- Reusable `JobCard` component
- `JobList` component for rendering multiple jobs
- Conditional rendering
- TypeScript props typing
- Responsive UI structure

---

# 🛠️ Technologies Used

- React
- TypeScript
- CSS
